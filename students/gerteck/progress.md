### Lead & Technical Contributions

* **TypeScript & ESM Migration:** Leading the effort to modernize the codebase by migrating the CLI and Core package patches to TypeScript and transitioning CLI output from CommonJS to ES Modules.
* **Architectural Refactoring:** Improved the maintainability of the Core package by refactoring the `Site` class into a Facade pattern, separating logic into dedicated Managers.
* **Infrastructure & Tooling:** Managed critical dependency updates (Node v22, TypeScript v5, Lerna, Jest) and established project guidelines by introducing `AGENTS.md` and improving release documentation.
