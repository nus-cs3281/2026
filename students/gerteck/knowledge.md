* **Claude Code** (CLI), **AntiGravity (Gemini)**, explored the use of AI to automate tasks and use **AI Agents** autonomous entities capable of executing multi-step developer workflows like codebase searching, editing, and testing.

  * Used to come up with proof-of-concepts (e.g. integration of specific library) and prototyping.

### Copilot Code Review

* **GitHub Copilot for Code Review**: investigated and used it to assist in my role as a lead developer. I focused on using AI to generate PR summaries and identify logic flaws or style inconsistencies to streamline the manual review process.
